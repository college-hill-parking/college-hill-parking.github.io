About the Project

This project was created as part of an English 402 class to brainstorm ways to help people navigate College Hill parking, which has long been a challenge in Pullman. Many visitors receive tickets for unknowingly parking in restricted zones. With fines increasing, it's a costly mistake.

Our idea was to build this website to make parking rules clearer and help visitors more easily find legal spots, especially during WSU gamedays.

As our class project ends, we hope future students or community members can expand on what we've created. The project is open source and hosted on GitHub: View Repository . Instructions are available on GitHub for forking the repo or exploring the codebase.

The maps on this site were created using Google My Maps, and they can be copied and modified for future use.
